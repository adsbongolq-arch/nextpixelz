# 🎨 VISUAL IDENTITY & COLOR STRATEGY: NEXTPIXELZ

## 1. Strict Palette
Utilize the logo's color combination as the site's design foundation.
*   **Background (bg-primary):** `#1D1B4D` (Dark Navy Blue) - A full-page dark mode experience.
*   **Highlights & CTA (bg-accent, text-accent):** `#F0712C` (Orange) - Used for buttons, active links, and key stats.
*   **Text:** Light off-white (`#E0E0E0`) for maximum readability against dark background.

## 2. Design DNA
*   **Geometry:** Incorporate the triangle patterns from the logo into section dividers, card backgrounds, and button hover states.
*   **Glassmorphism:** Use translucent (`bg-opacity-80`) panels with frosted glass effect for service cards and dashboard elements.

## 3. Animation Directive
*   **Triangle Pulse:** Add a subtle, low-opacity triangle pulse effect in the Hero background using GSAP.
*   **Accent Glow:** Add a soft orange glow (`shadow-accent-glow`) to CTA buttons on hover.
